# Retrieve variable PSI event IDs

Returns the event IDs stored by
[`FindVariablePSIEvents`](https://avisrilab.github.io/Matisse/reference/FindVariablePSIEvents.md).
Returns an empty character vector if variable events have not been set.

## Usage

``` r
VariablePSIEvents(object)
```

## Arguments

- object:

  A `MatisseObject`.

## Value

Character vector of variable event IDs.
