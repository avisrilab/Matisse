# Get exclusion read count matrix

Get exclusion read count matrix

## Usage

``` r
GetExclusionCounts(object, ...)

# S4 method for class 'MatisseObject'
GetExclusionCounts(object, ...)
```

## Arguments

- object:

  A `MatisseObject`.

- ...:

  Additional arguments (unused).

## Value

A sparse matrix (cells x events) of exclusion read counts.
