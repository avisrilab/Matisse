# Get raw junction count matrix

Get raw junction count matrix

## Usage

``` r
GetJunctionCounts(object, ...)

# S4 method for class 'MatisseObject'
GetJunctionCounts(object, ...)
```

## Arguments

- object:

  A `MatisseObject`.

- ...:

  Additional arguments (unused).

## Value

A sparse matrix (cells x junctions) of read counts.
