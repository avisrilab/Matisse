# Get inclusion read count matrix

Get inclusion read count matrix

## Usage

``` r
GetInclusionCounts(object, ...)

# S4 method for class 'MatisseObject'
GetInclusionCounts(object, ...)
```

## Arguments

- object:

  A `MatisseObject`.

- ...:

  Additional arguments (unused).

## Value

A sparse matrix (cells x events) of inclusion read counts.
